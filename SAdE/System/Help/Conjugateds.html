Roteiro de Diários

  • Criar as Turmas Conjugados.
    • O sistema deve criar uma turma conjugado (a principal), usando o curso da disciplina.
    • Use 'Adicionar Turma Conjugada' para criar mais turmas.
    • Na tabela de turmas conjugadas, use o link 'Alunos' para adicionar alunos para a turma.
  • Define seus dias de Aula e conteúdos, usando o link 'Aulas' no menu horizontal acima.
  • Cadastre freqüência dos alunos, usando o link 'Freqüências' no menu horizontal acima.
  • Cadastre avaliações (N1,N2,..), usando o link 'Avaliações' no menu horizontal acima.
  • Cadastre notas dos alunos, usando o link 'Notas' no menu horizontal acima.
  • Imprime Aulas & Conteúdos, Freqüências e Notas, usando o link 'Impressos' no menu horizontal acima.
