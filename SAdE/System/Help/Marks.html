Cadastrar Notas

  • As notas lançadas aparecerão na tabela dos acadêmicos abaixo.
  • Cadastre somente notas com valor entre 0 e 10.
  • Para 'descadastrar' uma nota, digite um traçinho (-) no lugar da nota.
  • Para lançar mais avaliações, use o link 'Avaliações' no menu acima.
  • Para uma versão imprimível (PDF) das notas, use o link 'Imprimir Notas' no menu acima.
  • A média final é calculado como a soma das notas individuais vezes o peso, dividido com a soma dos pesos.
